# Auralis

Auralis is a self-hosted Android recovery and automation system.

## Main products

- **Auralis Protect**: Android protection app installed on the secured phone.
- **Auralis Control Mobile**: Android controller/finder app for trusted phones.
- **Auralis Control Desktop**: Desktop controller app for Windows/macOS/Linux, planned later.
- **Auralis One**: Combined app planned after Protect and Control are stable.

## Current priority

Build **Auralis Protect** first.
